Renesas Envision Kit RX65N ガジェット
=========
![R5F564ML](../RTK5RX65N.jpg)

## 概要
Envision Kit RX65N LCD 表示サンプル
   
## プロジェクト・リスト
 - main.cpp
 - Makefile
   
## ハードウェアーの準備
 - SD カードインターフェースの準備
   
## リソースの準備
 - 漢字フォントをSDカードに準備「graphics/kfont16.bin」を SD カードのルートに置く。
   
## ビルド方法
 - make する。
 - lcd_sample.mot ファイルを書き込む。
